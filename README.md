# Digital Garden

A minimalist digital garden built with Astro and Starlight.

## Design Philosophy

This garden follows **Modern Minimalism** principles:
- **Typography**: Inter for UI, Geist Mono for code
- **Colors**: Pure white (#ffffff) in light mode, near black (#0a0a0a) in dark mode
- **Aesthetic**: Clean sidebar and header with blur effects

## Structure

- **Stream** (`src/content/docs/stream/`) — Ephemeral thoughts and notes
- **Wiki** (`src/content/docs/wiki/`) — Evergreen knowledge and reference
- **Meta** (`src/content/docs/meta/`) — About the garden itself

## Development

```bash
npm install

```

## Deploy
