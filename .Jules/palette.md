## 2026-03-18 - Semantic Navigation and Focus States for Tag Lists
**Learning:** Generic wrappers (`<div>`) for lists of navigation links (like tags) provide poor context for screen reader users. Additionally, relying solely on hover states for interactive tag elements severely diminishes keyboard accessibility, especially in dark mode where default focus rings may blend into the background.
**Action:** Always wrap grouped navigational links (like page tags) in a semantic `<nav aria-label="...">` element. Ensure each link has a descriptive `aria-label` (e.g., `aria-label="View all pages tagged with ${tag}"`) to clarify its destination. Furthermore, explicitly implement `focus-visible` utility classes (e.g., `focus-visible:ring-2`, `dark:focus-visible:ring-offset-stone-900`) tailored to both light and dark modes to guarantee visible focus indicators during keyboard navigation.
