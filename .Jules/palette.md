## 2024-05-15 - [Astro Tag Components Accessibility & Deep Linking]
**Learning:** When displaying interactive tag taxonomies, elements like `PageTags.astro` should use semantic `<nav aria-label="Page tags">` wrappers instead of plain `<div>`s for improved screen reader context. Furthermore, providing deep-linking functionality (e.g., `#tag-${encodeURIComponent(tag)}`) paired with explicit focus management via utility classes (`focus-visible:ring-2`, `focus-visible:ring-primary`) ensures an accessible, keyboard-navigable experience that works in both light and dark mode.
**Action:** Always wrap tag collections in semantic `<nav>` elements with appropriate ARIA labels, ensure proper URI encoding when dynamically generating href anchors, and apply explicit `focus-visible` offset ring styles to preserve clarity across themes.
