## 2024-03-24 - Semantic and Accessible Page Tags
**Learning:** Utilizing semantic HTML tags (`<nav aria-label="Page tags">`) alongside appropriate ARIA attributes for dynamically generated links significantly improves context for screen reader users. Furthermore, deep linking tag elements directly to their section within a tag cloud page (e.g., `#tag-{slug}`) eliminates redundant navigation steps. Integrating customized Tailwind focus states (e.g., `focus-visible:ring-2`, `focus-visible:ring-primary`) ensures predictable and visible interaction for keyboard-only navigation.
**Action:** Always wrap collections of tag-like links in `<nav>` wrappers with clear `aria-label`s, ensure individual items provide clear context via `aria-label`, and mandate consistent keyboard focus states to prevent focus loss during navigation.
